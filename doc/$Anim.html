$Anim 方法

介绍：返回一个Fx的实例(动画对象)

方法：

  • Fx $Anim(Element el,[Object events])

Fx对象

介绍：提供一些动画方法，可以灵活地构建复杂的动画效果

方法：

  • Fx add(Action act)
  • Fx start()
  • Fx restart()
  • Fx delay(Number time)
  • Fx pause()
  • Fx resume()
  • Fx stop(String pos)
  • ArrayActions getQueue()
  • Element getElement()
Fx add(Action act)

为Fx添加一个动作。通过链式方法可以同时添加多个动作。

Action 动作对象的格式为：


var act = {
    to:{                                //动画目标样式  可略
        left:500,
        top:400,
        opacity:0.8,
        width:150,
        height:30
    },
    onQueueStart:function(fx,el){},    //动作开始时事件 可略
    onQueueComplete:function(fx,el){}, //动作结束时事件 可略
    transition:easing.linear,          //动作哈希渐变方法 可略
    delay:500,                         //执行动作前等待时间(毫秒) 可略
    duration:2000,                     //执行动作所需时间(毫秒) 可略
};

Fx start()
开始动画
Fx restart()
重新开始动画
Fx delay(Number time)
等待一定时间开始动画
Fx pause()
暂停动画
Fx resume()
恢复暂停继续动画
Fx stop(String pos)
停止动画 参数：stop("start")停止到开始位置；stop("end")停止到结束位置；stop("astart")停止到当前动作的开始位置；stop("aend")停止到当前动作的结束位置；
ArrayActions getQueue()
获取该动画的动作队列(数组)
Element getElement()
获取动画的标记

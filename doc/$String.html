String trim(String str)

同时去除字符串两端的空格

如："a"+$String.trim("  test   ")+"a"; //"atesta"

String escapeHTML(String str)
格式化HTML 如 $String.escapeHTML("<"); //&lt;
String unescapeHTML(String str)
反格式化HTML 如 $String.escapeHTML("&lt"); //<
Nunmber byteLength(String str)
取得字符的字节长度，汉字认为是两个字符
String delLast(String str)
除去最后一个字符
Nunmber toInt(String str)
转换成整型数字
Array toArray(String str,[String split])
转换成数组
String left(String str,Int n)
取左边多少字符，中文两个字节
String right(String str,Int n)
取右边多少字符，中文两个字节
String removeHTML(String str)
除去HTML标签
String format(String str)
模版函数 如： $String.format("<div>{0}</div>{1}","test1","test2"); // <div>test1</div>test2
String toLower(String str)
将字符串类所有字母转换为小写
String toUpper(String str)
将字符串类所有字母转换为大写
String on16(String str)
将字符串转换成16进制表示
String un16(Stringstr)
对escape()函数处理的字符串解码

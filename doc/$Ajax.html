$Ajax 对象

介绍：对原生的XHR封装

方法：

  • void jsonp(String url,Object|String data,Function callback)
  • void get(String url,[Object|String data,[Function callback],[String format]])
  • void post(String url,[Object|String data,[Function callback],[String format]])
  • void ajax(String url,Object options)
void jsonp(String url,Object|String data,Function callback)
用于jsonp跨域传输
void get(String url,[Object|String data,[Function callback],[String format]])
get方式的ajax请求
void post(String url,[Object|String data,[Function callback],[String format]])
post方式的ajax请求
void ajax(String url,Object options)
底层 AJAX 实现。简单易用的高层实现见 $Ajax.get, $Ajax.post 等。$Ajax.ajax() 返回其创建的 XMLHttpRequest 对象。大多数情况下你无需直接操作该函数，除非你需要操作不常用的选项，以获得更多的灵活性。 


    $Ajax.ajax(url,{
      method:     'get',
      async:      true,
      data:       null,
      format:     'text',
      encode:     'UTF-8',
      success:   function(){},
      failure:   function(){},
			whatever: function(){}
    });

  • void ready(Function fn)

  • mix prop(Element el,String name,[mix value])

Element $id(String id)
类似于document.getElementById
ArrayElements $name(String name,[Element parent])
类似于document.getElementsByName
ArrayElements $tag(String tagName,[Element parent])
类似于document.getElementsByTagName
ArrayElements $class(String className,[String tag, [Element parent]])
通过class类名获得标记，类似于document.getElementsByClassName
ArrayElements $attr(String search,[Element parent])

通过标记属性值获得标记

  • [Attr] 或者tagName[Attr] 选择具有此属性的元素 blockquote[title]
  • [Attr="value"] 或者tagName[Attr="value"] 选出属性值精确等于给出值的元素 h2[align="left"]
  • [Attr^="value"] 或者tagName[Attr^="value"] 选出属性值以给出值开头的元素 h2[align^="right"]
  • [Attr$="value"] 或者tagName[Attr$="value"] 选出属性值以给出值结尾的元素 div[class$="vml"]
  • [Attr*="value"] 或者tagName[Attr*="value"] 选出属性值包含给出值的元素 div[class*="grid"]
  • [Attr~="value"] 或者tagName[Attr~="value"] 原元素的属性值为多个单词，给出值为其中一个。 li[class~="last"]
  • [Attr|="value"] 或者tagName[Attr|="value"] 原元素的属性值等于给出值，或者以给出值加“-”开头 span[lang|="en"]
  • [Attr!="value"] 或者tagName[Attr!="value"] 反选属性值选择器

Element $one(String selector,[Element parent])
通过CSS选择器获得标记，只返回第一个匹配的元素，不支持IE8以下浏览器
ArrayElements $all(String selector,[Element parent])
通过CSS选择器获得标记，只返回所有匹配的元素，不支持IE8以下浏览器
void ready(Function fn)
添加方法，当Dom加载完成执行，可以添加多个方法
mix attr(Element el,String name,[mix value])
获取[设置]标记HTML的属性
Element removeAttr(Element el,String name)
删除标记的指定属性
mix prop(Element el,String name,[mix value])
获取[设置]标记DOM对象指定的的属性
String[void] css(Element el,String name,[mix value])
获取[设置]标记指定的CSS样式
Element removeCss(Element el,String name)
删除标记指定的CSS样式
String[void] text(Element el,[mix value])
获取[设置]标记的文本(节点)
String[void] html(Element el,[mix value])
获取[设置]标记的innerHTML
String[void] val(Element el,[mix value])
获取[设置]标记的值
Element show(Element el,[mix display])
显示标记，可以设置以什么方式显示(参照display样式)
Element hide(Element el)
隐藏标记
Element toggle(Element el)
切换显示/隐藏标记
Boolen hasClass(Element el,String name)
返回标记是否有指定的class样式
Element addClass(Element el,String name)
为标记添加指定的class样式
Element removeClass(Element el,String name)
为标记删除指定的class样式

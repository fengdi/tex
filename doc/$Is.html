$Is 对象

介绍：提供一些对类型的检测的方法

方法：

  • Boolen String(mix m)
  • Boolen Function(mix m)
  • Boolen Object(mix m)
  • Boolen Number(mix m)
  • Boolen Array(mix m)
  • Boolen EmptyObject(mix m)
  • Boolen PlainObject(mix m)
  • String getType(mix m)
Boolen String(mix m)
检测是否为字符串
Boolen Function(mix m)
检测是否为方法
Boolen Object(mix m)
检测是否为对象
Boolen Number(mix m)
检测是否为数字
Boolen Array(mix m)
检测是否为数组
Boolen EmptyObject(mix m)
检测是否为空对象（不包含任何属性）。如：$Is.isEmptyObject({}) // true $Is.isEmptyObject({ foo: "bar" }) // false
Boolen PlainObject(mix m)
测试对象是否是纯粹的对象（通过 "{}" 或者 "new Object" 创建的）。如：$Is.isPlainObject({}) // true $Is.isPlainObject("test") // false
Boolen String(mix m)
返回该类型的字符串。 如： $Is.getType(190); //Number
